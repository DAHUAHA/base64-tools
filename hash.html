What is a Hash?

A hash function converts input data into a fixed-size string of bytes. The output (digest) appears random but is deterministic — the same input always produces the same output. Hashes are one-way: you cannot reverse a hash to get the original input. They are used for password storage, data integrity verification, and digital signatures.
