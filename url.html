常见问题

URL 编码和普通编码有什么区别？
URL 编码专门处理 URL 保留字符（&、=、? 等）和非 ASCII 字符，确保链接不被截断解析。其他编码如 UTF-8 用于文本文件。
为什么复制粘贴的链接有时候打不开？
可能是链接中的特殊字符没有编码。比如"你好"应该是"%E4%BD%A0%E5%A5%BD"，直接粘贴会出问题。
encodeURI 和 encodeURIComponent 有什么区别？
encodeURI 保留 URL 结构字符（:、/、? 等），不编码；encodeURIComponent 全部编码。参数值用后者，整条 URL 用前者。
中文 URL 路径怎么转？
把整段中文路径粘贴，工具会输出正确的百分号编码。浏览器地址栏会自动编码，但程序调用时需要手动处理。
怎么批量解码多个 URL？
粘贴多行 URL，一次性批量编解码，适合处理爬虫抓取的链接列表。
